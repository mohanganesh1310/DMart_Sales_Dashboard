#  D-Mart Sales Analysis Dashboard | Power BI + SQL

Developed an interactive D-Mart Sales Analysis Dashboard using Power BI connected with SQL database to analyze sales performance and business KPIs.

# Dashboard Highlights:
• Revenue & Profit Analysis (Daily, Monthly, Quarterly, Yearly)
• Sales Trend Analysis
• Product Category Performance
• Top 10 Products Analysis
• Store-wise Profitability Insights
• Orders & Units Sold Tracking
• Interactive Filters and Drilldowns

# Tools & Technologies:
• SQL
• Power BI
• DAX

# Key Insights:
• Identified top-performing products and profitable stores
• Analyzed revenue and profit trends over time
• Compared category-wise sales performance
• Built dynamic and interactive visualizations for business decision-making

This project helped me strengthen my skills in SQL querying, business intelligence, data visualization, and KPI analysis.
